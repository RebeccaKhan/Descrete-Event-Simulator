Task 3.3 Analysis and General Questions¶

Task 3.3.1: Seed for Random Number Generation¶

Simulating non-deterministic system can still be considered as running an 'experiment'. The seed to the RNG in the non-deterministic system allows us to check the repeatability and reproducibility of our 'experiment'.Even though this causes the RNG to not be truly random, a large number of simulation runs should emulate the randomness of the real world for our experiment.

Task 3.3.2 Bins for Histogram¶

There exists no definite rule for the number of bins to be used for a histogram. A common 'rule of thumb' is that if the number of samples is large enough, the number of bins should be roughly the square root of the number of samples. If we decrease the number of bins, we get a more 'block like' distribution which does not have the sensitivity to exactly depict the curves required in distributons like exponential or gaussian. If we increase the number of bins, we start seeing the granularity in the distribution and it has more 'peaks and valleys' than necessary. We will also have empty spaces if no values from our sample falls inside an unnecassarily small bin

Task 3.3.3 System Utilization¶

From the simulation results of Task 3.2.2, we see that for lower 'Theoretical' values of RHO the observed values of RHO which is the utilization have some deviation. This deviation increases as we increase the Expected RHO. This behaviour is due the the finite queue size (or buffer size which causes a non-zero blocking probability). As our utilization increases, the server will be busy for more and more time and the variance in the arrival and departures will cause some packets to drop. This means that in the long run, our mean utilization will be lesser than expected as the dropped packets will never reach the server and hence will play no role in the departures. In other words, the while our arrival and departure rates are fixed, our arrived packets will be more than the departed packets and not the same and hence the observed utilization will be less than the theoretical one. If we increase the simulation time to infinty, we will still have a non-zero blocking probability, but the variance of it will decrease, that is it will reach a steady state and hence we will get an exact correlation between the blocking probability and the deviation from expected values ( the two lines from the figure from Task 3.2.2 will overlay on each other). If we have an infinte S, we will have a zero blocking probability and hence our observed values of utilization will equal the theoretical one ( just as we see in the figure for very low utilization values, the blocking probability is zero and deviation is also zero).
